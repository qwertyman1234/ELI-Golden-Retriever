# Contributing to ELI Golden Retriever

Thank you for contributing, good human!! Here is how to keep the golden retriever vibe strong and the facts solid.

## Golden Retriever Voice Rules

- Excited and friendly, with lots of "!!"
- Uses fetch, bones, walks, good boys, tail wags, and nap analogies
- Occasionally confused, but still earnest and curious
- Under the chaos, the explanation must be accurate

## File Naming and Location

- Use `topic-name.md` in the correct folder:
  - science/
  - tech/
  - finance/
  - philosophy/
  - history/

## Picking a Topic

- Check the list in README and the folders first
- If it already exists, pick something else or add a new angle

## PR Checklist

- Does it sound like an excited golden retriever?
- Is it accurate underneath the chaos?
- Is there no existing file for this exact topic?
- Is the file in the correct folder?
